# MGTheme Extension Changelog

## Version 0.0.1 (Release Date: 30/11/2023)

### Features:
- Inizio dello sviluppo dell'estensione MGTheme per Visual Studio Code.
- Aggiunta di un tema predefinito chiamato "MG Light" con colori di base.

## Version 0.0.2 (Release Date: 30/11/2023)

### Features:
- Aggiornamento dei colori nel tema "MG Light".
- Introduzione di un nuovo tema scuro denominato "MG Dark" con colori di base.

## Version 0.0.3 (Release Date: 30/11/2023)

### Features:
- Sistemazione dei colori per migliorare la coerenza nel tema "MG Light".
- Aggiunta di opzioni di personalizzazione per alcuni elementi specifici.

## Version 0.0.4 (Release Date: 30/11/2023)

### Features:
- Ulteriori regolazioni dei colori in entrambi i temi.
- Risoluzione di piccoli problemi di contrasto e leggibilità.

## Version 0.0.5 (Release Date: 30/11/2023)

### Features:
- Aggiunta di nuovi colori e miglioramenti generali.
- Risoluzione di problemi segnalati dalla community.

## Version 0.0.6 (Release Date: 30/11/2023)

### Features:
- Introduzione di una modalità ad alta visibilità per migliorare l'accessibilità.
- Aggiornamento delle impostazioni predefinite.

## Version 0.0.7 (Release Date: 30/11/2023)

### Features:
- Miglioramenti delle performance.
- Correzione di bug relativi all'interfaccia utente.

## Version 0.0.8 (Release Date: 30/11/2023)

### Features:
- Ottimizzazione della gestione dei colori per una maggiore personalizzazione.
- Introduzione di opzioni avanzate di formattazione del codice.

## Note:
Questa versione continua a consolidare e migliorare l'esperienza d'uso dell'estensione MGTheme. Grazie per il vostro continuo supporto e feedback! 🚀
